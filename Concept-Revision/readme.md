## Overview

This repository contains well-structured Core Java revision material organized topic-wise for quick learning, exam preparation, and interview revision.

It covers all fundamental Object-Oriented Programming concepts, Java hierarchy relationships, and the Java Collection Framework with clean and minimal examples.

This repository is designed for strong concept clarity rather than large project implementation.
